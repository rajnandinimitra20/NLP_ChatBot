## DESCRIPTION
A PYTHON CHATBOT THAT ANSWERS FREQUENTLY ASKED QUESTIONS (FAQS) AND CAN LEARN NEW QUESTIONS FROM USERS IN REAL-TIME.  
IT’S INTERACTIVE, USER-FRIENDLY, AND STORES ALL LEARNED FAQS IN A CSV FILE FOR FUTURE USE.


## FEATURES
- ANSWERS PRE-DEFINED FAQS STORED IN `FAQ.CSV`.
- PROMPTS USERS TO TEACH IT ANSWERS FOR UNKNOWN QUESTIONS.
- STORES ALL LEARNED Q&A IN A CSV FILE FOR PERSISTENCE.
- HANDLES GREETINGS AND FAREWELLS.
- LIGHTWEIGHT: USES ONLY PYTHON’S STANDARD LIBRARIES (`CSV`, `STRING`, `OS`).


## INSTALLATION
1. CLONE THE REPOSITORY OR DOWNLOAD THE FILES.
2. ENSURE `CHATBOT.PY` AND `FAQ.CSV` ARE IN THE SAME FOLDER.
3. NO ADDITIONAL PACKAGES ARE REQUIRED — WORKS WITH PYTHON 3.X.


## USAGE
1. OPEN A TERMINAL OR COMMAND PROMPT IN THE PROJECT FOLDER.
2. RUN THE CHATBOT:

PYTHON CHATBOT.PY
